## 使用
1. 安装依赖
```Bash
npm i
```
2. 添加环境变量,需要配置3个变量
```Bash
# 页面标题
VITE_APP_TITLE = 'api测试平台'

# 开发环境配置
VITE_APP_ENV = 'development'

# 若依管理系统/开发环境
VITE_APP_BASE_API = 'http://127.0.0.1:4000'
```
3. 启动项目
```Bash
npm run dev
```
